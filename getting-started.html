Using Yeoman

Recommended!
Btapp.js is available via Twitter's Bower, which means it can be installed easily using Google's Yeoman, which is an awesome web scaffolding/testing/building tool build on top of Grunt.js

yeoman init //or `yeoman init backbone`
yeoman install btapp
yeoman install backbrace
While there are Btapp.js Yeoman generators in the works, for the time being you'll have to manually add links in your html to backbone.js and underscore.js (if you did the plain init), then also jstorage.js, btapp.min.js, and backbrace.js.

Javascript Interface

There is a single base class called Btapp. Create a new object, tell it to connect to the underlying Torque torrent client and you're off to the races. 

var btapp = new Btapp();
btapp.connect();
Ready to do something with the torrent client? Take a look at the real-time api visualizer or the code snippets page.
